## About The Project

The **Blogy Blog Website** is a simple yet powerful blogging platform template designed to share articles, news, tutorials, and stories online.  

It features a clean design with an emphasis on readability and user experience, making it perfect for personal blogs, tech blogs, travel diaries, or content-focused websites.  

Fully responsive and easy to customize, Blogy ensures your content looks beautiful on any device.

### Built With

- HTML5  
- CSS3

### Features

- Clean and modern UI design  
- Responsive layout for desktop, tablet, and mobile  
- Homepage with latest blog posts preview  
- Individual post pages with featured images  
- Categories and tags for organized content  
- Author section for personal branding  
- Contact and social media links integration
